# survey-next

This is a Next.js based upgrade to the original "Kea Survey Tool" first built in 2019.

## Getting Started

Copy the `.env.example` file to `.env` and fill out the environment variables accordingly.

To run the development server (from the repository root)

```bash
pnpm dev --filter=survey-next...
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
